= Spring Reactive Web Sample Project

Spring Reactive Web Sample Project based on link:https://start.spring.io/[Start Spring] (with

  - `Maven` and
  - `Spring Boot` version `2.0.0-SNAPSHOT` (`@170515`);
  - `g:de.mirb.pg.spring`;
  - `a:reactive-spring`;
  - dependencies: `Reactive Web`).
